# Military Muscle Workout App

A 5-day strength + boxing workout tracker built with React.

## Getting Started

```bash
npm install
npm start
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Features
- 5-day workout plan (Mon–Fri)
- Boxing combinations included
- Tap exercises to check them off
- Progress bar per day
- Color-coded by exercise type

## Build for Production
```bash
npm run build
```
